# 피파온라인 전적검색 사이트

### 개요
* **HTML**과 **JavaScript** ,**NEXON API**를 이용한 피파온라인4 전적검색 웹사이트 구현 

### 공헌한 내용
* 참여도: **개인프로젝트**
* HTML, CSS, BootStrap, JavaScript, jQuery
* Json, Nexon API
<br>
